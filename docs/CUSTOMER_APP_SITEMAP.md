# 고객 앱 사이트맵 및 네비게이션 가이드 🗺️

## 📱 전체 사이트맵

### 🏠 메인 구조
```
/customer/
├── 🔐 로그인 (/login)
├── 🏠 대시보드 (/dashboard) ⭐️ 홈
├── 🍽️ 음식 기록 (/food-diary)
│   ├── 📷 촬영 (/camera)
│   ├── ❓ AI 질문 (/questions/[sessionId])
│   ├── ✅ 결과 (/result)
│   └── 📊 통계 (/stats)
└── 📝 생활 기록 (/lifestyle)
```

---

## 🧭 네비게이션 시스템

### 1. 상단 헤더 (고정)
```
┌─────────────────────────────────┐
│ 🏥 명성약국    김○○님 | 로그아웃  │
└─────────────────────────────────┘
```
- **왼쪽**: 명성약국 로고 (터치 시 홈으로)
- **오른쪽**: 고객명, 로그아웃 버튼

### 2. 하단 네비게이션 바 (고정)
```
┌───────────┬───────────┬───────────┐
│ 🏠 홈     │ 🍽️ 음식기록│ 📝 생활기록│
└───────────┴───────────┴───────────┘
```
- **3개 주요 탭**으로 구성
- **현재 활성 탭** 하이라이트 표시
- 모든 주요 페이지에서 접근 가능

## 🔄 페이지별 네비게이션

### 🏠 대시보드 (/customer/dashboard)
**진입점**: 로그인 후 첫 화면
```
📍 현재 위치: 홈
┌─────────────────────────────────┐
│ 👋 안녕하세요, 김○○님!           │
│ 건강한 하루 되세요!              │
├─────────────────────────────────┤
│ 🍽️ 음식 기록  📝 생활 기록      │
├─────────────────────────────────┤
│ 📷 음식촬영  😴 수면기록  💊 복약 │
│              전체 기록 보기 →    │
├─────────────────────────────────┤
│ 📋 최근 상담 기록 (3개)         │
└─────────────────────────────────┘
```

**이동 경로**:
- `음식 기록` → `/customer/food-diary`
- `생활 기록` → `/customer/lifestyle`
- `음식 촬영` → `/customer/food-diary/camera`
- `상담 기록` → `/customer/consultation/[id]`

### 🍽️ 음식 기록 (/customer/food-diary)
**진입점**: 대시보드에서 음식 기록 카드
```
📍 현재 위치: 음식기록 (하단바 활성)
┌─────────────────────────────────┐
│ 음식 기록 📱           📷 촬영   │
├─────────────────────────────────┤
│ 📅 날짜 선택:      [2024-01-15] │
│ 📷 촬영  📊 통계  🏠 홈으로     │
├─────────────────────────────────┤
│ 📊 오늘 식사 기록 (3개)         │
│ ├ 🌅 아침: 토스트, 우유         │
│ ├ 🌞 점심: 김치찌개             │
│ └ 🌙 저녁: 치킨샐러드           │
└─────────────────────────────────┘
```

**이동 경로**:
- `촬영` → `/customer/food-diary/camera`
- `통계` → `/customer/food-diary/stats`
- `홈으로` → `/customer/dashboard`
- `개별 기록` → `/customer/food-diary/result?recordId=`

### 📷 음식 촬영 (/customer/food-diary/camera)
**진입점**: 음식 기록에서 촬영 버튼
```
📍 현재 위치: 음식 촬영
┌─────────────────────────────────┐
│ ← 음식기록   📷 음식 촬영     홈  │
├─────────────────────────────────┤
│ [실시간 카메라 뷰]              │
│                                │
│         📸 촬영 버튼             │
├─────────────────────────────────┤
│ 🧠 스마트 AI 분석               │
│ • 상황별 필요한 질문만          │
├─────────────────────────────────┤
│ 📷 촬영 팁                      │
│ • 충분한 조명 확보              │
└─────────────────────────────────┘
```

**이동 경로**:
- `음식기록` → `/customer/food-diary`
- `홈` → `/customer/dashboard`
- `분석 완료` → `/customer/food-diary/questions/[sessionId]`

## 🎨 모바일 UI/UX 특징

### 1. **터치 친화적 설계**
- 최소 44px 터치 영역 보장
- 명확한 버튼 라벨과 아이콘

### 2. **간결한 네비게이션**
- 3단계 이하 깊이 제한
- 언제든 홈으로 1-2터치로 이동 가능

### 3. **일관된 디자인 패턴**
```css
/* 주요 색상 */
- Primary: #6366f1 (인디고)
- Success: #10b981 (녹색)  
- Warning: #f59e0b (주황)
- Error: #ef4444 (빨강)

/* 간격 체계 */
- Small: 0.5rem (8px)
- Medium: 1rem (16px)
- Large: 1.5rem (24px)
```

### 4. **반응형 그리드**
- 2열: 메인 메뉴 카드
- 3열: 빠른 액션 버튼
- 1열: 리스트 항목

## 🚀 빠른 이동 경로

### 주요 작업별 최단 경로

**🎯 음식 촬영하기**:
```
홈 → 음식촬영 버튼 (1터치)
또는
홈 → 음식기록 → 촬영 (2터치)
```

**📋 기존 기록 보기**:
```
홈 → 음식기록 (하단바 또는 카드)
```

**🏠 홈으로 돌아가기**:
```
어디서든 → 🏠 홈 (하단바, 1터치)
```

**📊 통계 확인**:
```
홈 → 전체 기록 보기 → 통계 (2터치)
또는
음식기록 → 통계 (1터치)
```

## 🔧 개발자 참고사항

### 컴포넌트 구조
```
app/customer/
├── components/
│   ├── Navigation.tsx      # 상단헤더 + 하단바
│   ├── BackButton.tsx      # 뒤로가기 버튼
```

- 활성 탭 자동 하이라이트
