# 2026년형 약국 맞춤 인사·급여 관리 시스템 가이드

## 🎯 시스템 개요

**"클릭 3번으로 끝나는 급여 정산"**

매월 말일, 사장님은 다음 3번의 클릭만으로 급여 업무를 완료할 수 있습니다:
1. ✅ **승인** - 직원들이 입력한 근무 기록 확인 및 승인
2. ✅ **확정** - 급여 자동 계산 (Net→Gross 역산 포함)
3. ✅ **세무사 전송** - 급여대장 자동 생성 및 이메일 발송

## 🚀 핵심 기능

### 1. 세후(Net) → 세전(Gross) 자동 역산
이응창 약사님처럼 **"실수령액 700만원 고정"** 계약인 경우:
- 시스템이 2026년 4대보험, 소득세, 주민세를 역산
- 자동으로 세전 급여 계산 (약 927만원)
- 명세서에 명확히 표시

### 2. 2026년 세법 완벽 적용
- **국민연금**: 4.5% (상한 637만원)
- **건강보험**: 3.545%
- **장기요양**: 건강보험료의 12.81%
- **고용보험**: 0.9%
- **간이세액표**: 부양가족 수에 따른 구간별 계산

### 3. 비과세 자동 분리
- 식대: 월 20만원
- 자가운전보조금: 월 20만원
- 보육수당: 월 10만원
- **과세소득 = 총급여 - 비과세** 기준으로 세금 계산

### 4. 최저임금 자동 체크
- **2026년 기준**: 시급 10,030원, 월급 2,156,880원
- 설정 시 경고 알림으로 법 위반 방지

### 5. 포괄임금제 규제 대응
- 명세서에 **"고정연장근로수당 (월 O시간 분)"** 명확히 표기
- 단순 "직책수당"이 아닌 법적 요구사항 준수

## 📱 직원용 기능 (모바일 최적화)

### 화면 1: 근무 캘린더 (`/employee-hr/work-calendar`)

**상단 예상 수령액**
```
이번 달 예상 수령액
2,350,000원 (세후)
실시간 업데이트
```

**달력 뷰**
- 근무한 날짜 색상 표시
  - 🟢 초록색: 정상 출근
  - 🔵 파란색: 휴가
  - 🔴 빨간색: 결근
- 각 날짜에 근무시간 표시
- 연장근무 시간 별도 표시 (1.5배 아이콘)

**근무 입력 (+버튼)**
```
날짜: 2026-01-10 (자동 지정)
시작 시간: 09:00 (스크롤 선택)
종료 시간: 18:00
구분:
☐ 연장근무 (1.5배)
☐ 야간근무 (1.5배)
☐ 휴일근무 (2.0배)
메모: [선택사항]
```

**특징**
- 파트타임: 시간만 입력 → 시급 자동 계산
- 정규직: 초과근무만 별도 입력
- 입력 즉시 예상 수령액 업데이트

## 🔧 시스템 설정

### 1. 직원 등록 시 설정 항목

**기본 정보**
- 이름, 전화번호, 이메일
- 입사일, 주민번호 (암호화 저장)

**급여 계약 형태** ⭐
```
○ Gross (세전 계약)
   기본급: 3,500,000원
   
● Net (세후 계약)
   실수령액: 7,000,000원
   → 시스템이 세전 자동 역산
```

**부양가족 수**
```
부양가족: [1]명
→ 간이세액표 적용에 사용
```

**비과세 항목**
```
식대: 200,000원 (최대 20만원)
자가운전: 200,000원 (최대 20만원)
보육수당: 100,000원 (최대 10만원)
```

**고용 형태**
```
○ 정규직 (월급제)
● 파트타임 (시급제)
   시급: 15,000원
○ 계약직
```

### 2. 포괄임금제 설정 (선택)

```
고정 연장근로 시간: [20]시간
고정 연장근로수당: [450,000]원

→ 명세서 자동 표기:
  "고정연장근로수당 (월 20시간 분): 450,000원"
```

## 💡 활용 시나리오

### 시나리오 1: 정규직 약사 (Net 계약)

**설정**
```
이름: 이응창
계약: Net (세후 700만원)
부양가족: 2명
식대: 20만원
```

**매월 자동 처리**
1. 시스템이 700만원 실수령을 위한 세전 급여 역산
2. 2026년 세율 적용하여 약 927만원 계산
3. 명세서에 역산 과정 명시
4. 승인 → 확정 → 세무사 전송

### 시나리오 2: 파트타임 (시급제)

**설정**
```
이름: 정하영
고용형태: 파트타임
시급: 15,000원
```

**매월 처리**
1. 직원이 캘린더에 근무시간 입력
2. 시스템이 시급 × 시간 자동 계산
3. 4대보험 자동 공제 (주 15시간 이상 시)
4. 실수령액 자동 계산

### 시나리오 3: 정규직 + 초과근무

**설정**
```
이름: 함민
기본급: 350만원
포괄임금: 고정OT 20시간
```

**이번 달 근무**
```
정상 근무: 176시간
초과 근무: 30시간
- 고정OT: 20시간 (포괄임금에 포함)
- 추가OT: 10시간 (1.5배 수당 별도)
```

**계산 결과**
```
기본급: 3,500,000원
고정OT수당: 450,000원 (월 20시간)
추가OT수당: 300,000원 (10시간 × 1.5배)
────────────────
총 급여: 4,250,000원
```

## 🚨 주의사항

### 1. 초기 설정 시
- **계약 형태(Net/Gross) 정확히 설정**
  - 잘못 설정 시 급여 계산 오류
- **부양가족 수 정확히 입력**

- **변동급 입력 확인**
  - 상여금, 특별수당 등

### 3. 세무사 발송 전
- **급여대장 내용 최종 검토**
- **이메일 주소 확인**
- **암호 별도 전달**
